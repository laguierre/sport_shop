# Sport Shop Flutter App

A short demo of a Sport Shop Flutter application, using Hero and Transform widget.

## Getting Started

**Packages used:**

- flutter_svg: [link](https://pub.dev/packages/flutter_svg)
- font_awesome_flutter: ^9.2.0 [link](https://pub.dev/packages/font_awesome_flutter)
- smooth_page_indicator: ^1.0.0+2 [link](https://pub.dev/packages/smooth_page_indicator)
- Icons from FlatIcon. [link](https://www.flaticon.es/)

firebase_messaging: ^10.0.8
flutter_local_notifications: ^9.0.0
firebase_core: ^1.7.0
